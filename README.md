# Prerequisites
Nodejs 20

Ollama (with model llama3.2:latest)
>ollama pull llama3.2:latest

# 1. Install
Navigate to the directory with `app.js`, open terminal
>npm install

Navigate to `MCP Server` directory, open terminal
>npm install

>node app.js
